# 案例教学

第一步：构建 recipe_box 的专案项目

```
mkdir workspace
cd workspace
rails new recipe_box
cd recipe_box
git init
git add .
git commit -m "initial commit"
```
第二步：查看 recipe_box 的专案项目

rails s
